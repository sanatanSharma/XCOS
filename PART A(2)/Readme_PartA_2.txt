Name : Student Ranking System
Version : 1.0
Description : Generates the ranks of students according to pre-defined criterion based on their marks.

System Requirements : A java installed machine.

Instructions : Compile all the files and run StudentRanking.java.
               Enter the input as specified in the header in StudendRanking.java file.
               

File list and description : 
                          "StudentRanking.java" - File containing main method.
                          "StudentRankingSystem.java" - Contains the class "StudentRankingSystem" which has the logic for sorting 
                                                        and storing "Student" objects.
                          "Student.java" - Contains the class "Student" with get and set methods.
